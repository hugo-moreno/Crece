# 🚀 Escuela Virtual - Full Stack

### 🛠️ Instalación rápida
1. **Base de Datos:** Importar el archivo `.sql` en SQLyog.
2. **Backend:**
   - `cd Backend`
   - `npm install`
   - Configurar `.env` con tu clave de MySQL.
   - `node server.js` (Corre en puerto 50644)
3. **Frontend:**
   - `cd Frontend`
   - `npm install`
   - `npm run dev` (Corre en puerto 5173)
